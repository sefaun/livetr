## Installation

```sh
npm install
```

## Development

Open 2 terminal

```sh
npm run dev
```

wait for development host to run

```sh
npm run electron
```

## Setup

```sh
npm run build
```

change `"type": "module"` to `"type": "commonjs"` in **package.json**

```sh
npm run setup
```
